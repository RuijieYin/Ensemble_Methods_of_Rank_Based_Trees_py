# Python package for implementing ensemble methods of rank-based trees
* A R version can be found at: https://github.com/RuijieYin/Ensemble_Methods_of_Rank_Based_Trees


## Getting Started

### Dependencies

#### For Python package:

* Prerequisites: pandas, sklearn, numpy, shap-hypetune, lightgbm
* Python (>= 3.6.0)

### Installing

#### For Python package:

* Download the Python package from /dist/EnsembleRankTrees-0.1.1.tar.gz
* pip install EnsembleRankTrees
